# Instructions for Agents working on the OBP-MCP repo

## Overview
This is a Model Context Protocol server for the Open Bank Project (OBP-API) written with fastmcp. 

## Project Management
The project is managed with uv
commands should be prefaced with `uv run`

## Adding Dependencies
If you discover a dependency is missing, you can add it with `uv add *dependecy*`

## Tests
Write tests in pytest, in the /test folder
run tests with `uv run pytest` etc.
