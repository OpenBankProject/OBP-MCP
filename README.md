# OBP-MCP
An MCP Server for the OBP API

## Setup

Install [uv](https://docs.astral.sh/uv/) on your machine:

### MacOS/Linux
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### Windows
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```
